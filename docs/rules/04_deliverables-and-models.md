# 04. 成果物・モデル・出力フォーマット

## 🤖 モデル使い分け
- **計画・分析・高度な論理推論** → Opus
- **実行・サブタスク・その他大部分のステップ** → Sonnet

## 📦 成果物の報告ルール
- 成果物ファイルは **GitHub上のリンクをハイパーリンクで報告**（厳守）
- **ワンクリックで直接開ける**リンク形式
- 例: `[ファイル名](https://github.com/KazuyaMurayama/リポジトリ名/blob/main/パス/ファイル名)`

## 📄 出力フォーマット
- **md ファイルを優先**
- Googleスプレッドシート、Word、PPT は **ユーザー指示があれば**使用
